编译原理-目标代码生成

子不语

目标代码生成

RISC-V

寄存器

  • zero：存放常数0，不允许修改

  • ra：存放返回地址

  • t0~t6：保存临时变量

  • pc：保存下一条指令的地址

指令

li指令是一条伪指令，最终会翻译为addi t0, zero, 20的形式

  • 负数则为addi t0, zero, -20
  • 没有subi指令

系统调用

  1. 把服务号写入a7寄存器
  2. 把参数写入a0、a1、a2、a3、fa0、...
  3. ecall
  4. 从结果寄存器中获得返回值

代码段

  • .开始的指令是写给汇编器的指令

  • .data存储全局变量

    • g: .word 0
    • msg: .string "sss"以\0结尾的string
    • msg: .ascii "sds"的string不以\0结尾

  • .text存储运行代码

    • la t0, g # la: load address
    • lw t0, 0(t0) # lw: load word
    • sw t6, 0(t0) #sw: store word

默认在程序开始添加.text

  • 打印字符串：
  • mv
  • 数组：偏移地址
