3_定积分

雄关漫道真如铁，而今迈步从头越。

3_定积分

1. 定积分

定积分

  1. 区间\([a,b]\)的一个划分\(P\)是满足\(a=x_0\leq x_1\leq...\leq x_n=b\)的有限点集\(x_0,...,x_n\)，\(\Delta x_i=x_i-x_{i-1}\)。
  2. 设\([a,b]\)上的有界实函数\(f\)对\(P\)有\(M_i=\sup f(x),x\in[x_i-1,x_i]\)，\(m_i=\inf f(x),x\in[x_i-1,x_i]\)，称上积分为\(\overline{\int_a^b}fdx=\inf\{\sum M_i\Delta x_i\}\)，下积分为\(\underline{\int_a^b}fdx=\sup\{\sum m_i\Delta x_i\}\)。
  3. 若上下积分相等，称\(f\)在\([a,b]\)上可积，记为定积分\(\int_a^bfdx=\overline{\int_a^b}fd x=\underline{\int_a^b}fdx\)。

定积分的蕴含性质：

  1. 积分线性：

    \(\int_a^b(f(x)+g(x))dx=\int_a^bf(x)dx+\int_a^bg(x)dx\)

    \(\int_a^b𝜆f(x)dx=𝜆\int_a^bf(x)dx\)

  2. 积分区域的可加性：\(\int_a^cf(x)dx=\int_a^bf(x)dx+\int_b^bf(x)dx\)

  3. 积分值的保序性：设\(a<b\)，函数\(f\)和\(g\)在\([a,b]\)上可积并且满足\(f(x)\le g(x)\)，则有\(\int_a^bf(x)dx\le\int_a^bg(x)dx\)。

  4. 积分中值定理：

    设\(a<b\)，函数\(f\)在\([a,b]\)可积(则\(f\)在\([a,b]\)上有界)，若\(∀x∈[a,b]\)有\(m\le f(x)\le M\)，则\(m(b-a)\le\int_a^bf(x)dx\le M(b-a)\)。

    特别的，如果\(f\)在\([a,b]\)连续，那么存在\(c∈[a,b]\)，使得\(\int_a^bf(x)dx=f(c)(b-a)\)。

  5. 定积分存在条件：

    • 充分条件：

      • \(f(x)\)在\([a,b]\)上连续，则\(\int_a^b f(x)dx\)存在

      • \(f(x)\)在\([a,b]\)上单调，则\(\int_a^b f(x)dx\)存在

      • \(f(x)\)在\([a,b]\)上有界，且只有有限个间断点，则\(\int_a^b f(x)dx\)存在

    • 必要条件：函数有界。

定积分和微分

  1. 微积分基本定理：

    设函数\(f\)在\([a,b]\)连续，如果存在函数\(F\)，它在\([a,b]\)连续且可导，并且满足\(F’(x)=f(x)\)，那么函数\(f\)在\([a,b]\)可积，且\(\int_a^bf(x)\mathrm dx=F(b)-F(a)\)。称\(F\)为\(f\)的一个原函数。

    • 连续函数必有原函数。
    • 含有第一类间断点和无穷间断点的函数在包含该间断点的区间内必无原函数。

  2. 积分变量代换：即\(\mathrm dy=y'_x\mathrm dx\)

    设函数\(\varphi\)在\([\alpha,\beta]\)连续且一阶可导，\(\varphi(\alpha)=a\)，\(𝜑(𝛽)=b\)，\(\varphi((\alpha,\beta))\subset(a,b)\)，如果\(f\)在\([a,b]\)连续，那么\(\int_a^bf(x)\mathrm dx=\int_\alpha^\beta f(\varphi(t))\mathrm d\varphi(t)=\int_\alpha^\beta f(\varphi(t))\varphi’(t)\mathrm dt\)。

  3. 分部积分：即\((fg)'=f'g+g'f\)

    设函数\(u,v\in C^1[a,b]\)，则\(u(x)v(x)|_a^b=\int_a^bu(x)v’(x)\mathrm dx+\int_a^bv(x)u’(x)\mathrm dx\)。

反常积分

  1. 无穷区间的反常积分：\(F(x)\)为\(f(x)\)在相应区间的一个原函数。
    • \(\int_a^{+\infty}f(x)dx=\lim_{x\rightarrow+\infty}F(x)-F(a)\)，极限存在即收敛，否则发散；
    • \(\int_{-\infty}^bf(x)dx=F(b)-\lim_{x\rightarrow-\infty}F(x)\)，极限存在即收敛，否则发散；
    • \(\int_{-\infty}^{+\infty}f(x)dx=\lim_{x\rightarrow+\infty}F(x)-\lim_{x\rightarrow-\infty}F(x)\)，两个极限都存在即收敛，否则发散。
  2. 无界函数的反常积分：\(F(x)\)为\(f(x)\)在相应区间的一个原函数，瑕点为\(f(x)\)趋于无穷的点。
    • \(x=a\)为瑕点，\(\int_a^bf(x)dx=F(b)-\lim_{x\rightarrow a^+}F(a)\)，极限存在即收敛，否则发散；
    • \(x=b\)为瑕点，\(\int_a^bf(x)dx=\lim_{x\rightarrow b^-}F(b)-F(a)\)，极限存在即收敛，否则发散；
    • \(x=c\in(a,b)\)为瑕点，\(\int_a^bf(x)dx=\int_a^cf(x)dx+\int_c^bf(x)dx\)，两积分都存在则收敛，否则发散。
  3. 反常积分敛散性判别：利用积分的保序性，与收敛的反常积分进行比较。
  4. 重要结论：
    • \(\int_0^1\frac1{x^p}\begin{cases}收敛，0\lt p\lt 1\\发散，p\ge1 \end{cases}\)
    • \(\int_1^{+\infty}\frac1{x^p}\begin{cases}收敛，p\gt 1\\发散，p\le1 \end{cases}\)

曲线积分

曲线是一维的，因此无论是几维空间中的曲线都可以以单变量的参数方程表示。

设二维空间中曲线\(\begin{cases}x=x(t)\\y=y(t)\end{cases}\)，函数\(f:\mathbb R\times\mathbb R\rightarrow\mathbb R\)，\(L\)是曲线的一个光滑片段。

可以类似定积分地定义\(\int_L f\mathrm ds\)，称为\(f\)对\(L\)的曲线积分，其中\(\mathrm ds\)为\(L\)的弧段微元。

曲线积分是以参数为自变量的定积分，而定积分是在\(x\)轴的某段上进行的曲线积分。

  1. 由勾股定理，\((\mathrm ds)^2=(\mathrm dx)^2+(\mathrm dy)^2\)，于是\(\mathrm ds=\sqrt{(x'_t)^2+(y'_t)^2}\mathrm dt\)。
  2. 如果以\(x\)为参数，\(\mathrm ds=\sqrt{1+(y'_x)^2}\mathrm dx\)。
  3. 曲线积分\(\int_L f\mathrm ds=\int_L f\sqrt{(x'_t)^2+(y'_t)^2}\mathrm dt=\int_L f\sqrt{1+(y'_x)^2}\mathrm dx\)。

2. 定积分的计算

积分区域对称性

  1. \(f(x)\)为连续偶函数，则\(\int_{-a}^af(x)dx=2\int_0^af(x)dx\)。

  2. \(f(x)\)为连续奇函数，则\(\int_{-a}^af(x)dx=0\)。

  3. \(f(x)\)是周期为\(T\)的连续函数，则\(\forall a,\int_a^{a+T}f(x)dx=\int_0^Tf(x)dx\)。

  4. \(f(x)\)为连续函数，则\(\int_a^bf(x)dx=\int_a^bf(a+b-x)dx\)。

    • 常用于化简三角函数，也称区间再现：

      \(\int_0^\pi x\sin x\mathrm dx=\int_0^\pi(\pi-x)\sin(\pi-x)\mathrm dx=\int_0^\pi(\pi-x)\sin x\mathrm dx\)，于是

      \(\int_0^\pi x\sin x\mathrm dx=\frac\pi2\int_0^\pi\sin x\mathrm dx\)。

  5. 华莱士公式：

    \(\int_0^{\frac\pi2}\sin^nxdx=\int_0^{\frac\pi2}\cos^nxdx=\begin{cases}\frac{(n-1)!!}{n!!},n为大于1的奇数\\\frac{(n-1)!!}{n!!}\frac\pi2,n为正偶数 \end{cases}\)

    • \(\int_0^{\pi}\sin^nxdx=2\int_0^{\frac\pi2}\sin^nxdx\)
    • \(\int_0^{\pi}\cos^nxdx=\begin{cases}0,n为正奇数\\ 2\int_0^{\frac\pi2}\sin^nxdx,n为正偶数\end{cases}\)
    • \(\int_0^{2\pi}\cos^nxdx=\int_0^{2\pi}\sin^nxdx=\begin{cases}0,n为正奇数\\ 4\int_0^{\frac\pi2}\sin^nxdx,n为正偶数\end{cases}\)

变限积分

\(F(x)=\int_a^xf(t)dt\)为变上限的积分，同理也有变下限和上下限都变化的积分。

性质：

  • \(f(x)\)在\(I\)上可积，则\(F(x)=\int_a^xf(t)dt\)在\(I\)上连续。
  • \(f(x)\)在\(I\)上连续，则\(F(x)=\int_a^xf(t)dt\)在\(I\)上可导且\(F'(x)=f(x)\)。
  • 若\(x=x_0\in I\)是\(f(x)\)唯一的跳跃间断点，则\(F(x)=\int_a^xf(t)dt\)在\(x_0\)处不可导
    • \(F'_-(x_0)=\lim_{x\rightarrow x_0^-}f(x)\)
    • \(F'_+(x_0)=\lim_{x\rightarrow x_0^+}f(x)\)
  • 若\(x=x_0\in I\)是\(f(x)\)唯一的可去间断点，则\(F(x)=\int_a^xf(t)dt\)在\(x_0\)处可导
    • \(F'(x_0)=\lim_{x\rightarrow x_0}f(x)\)

求导公式：

  • 设\(F(x)=\int_{\varphi_1(x)}^{\varphi_2(x)}f(t)dt\)，则\(F'(x)=f(\varphi_2(x))\varphi_2'(x)-f(\varphi_1(x))\varphi_1'(x)\)。
  • 当\(f(t)\)中包含\(x\)，不能直接使用该求导公式，必须先通过换元等方式提出\(x\)。

变限积分和奇偶性：

  • \(f(x)\)为可积的奇函数，则\(\int_a^xf(t)dt\)为偶函数。
  • \(f(x)\)为可积的偶函数，则\(\int_0^xf(t)dt\)为奇函数。

有理函数的积分

\(f(x) = \frac{P(x)}{Q(x)}=\frac{a_0+a_1x+...+a_nx^n}{b_0+b_1x+...+b_mx^m}\)

其中\(P(x)\)和\(Q(x)\)都是实系数的多项式。若\(n<m\)，称\(f(x)\)为有理真分式，否则为有理假分式。

  1. 利用多项式的除法，可以把有理假分式表示为多项式和有理真分式的和。

  2. 有理真分式\(\frac{P(x)}{Q(x)}\)可以唯一地表示为以下形状的简单分式之和：

    \(\int\frac{dx}{(x-a)^n}\)和\(\int\frac{(Mx+N)dx}{(x^2+px+q)^n}\)

    该分解式称为真分式\(\frac{P(x)}{Q(x)}\)的简单分式分解或部分分式分解。

  3. \(\int\frac{dx}{(x-a)^n}=\begin{cases}\ln|x-a|+C,n=1\\-\frac{dx}{(n-1)(x-a)^{n-1}}+C,n\ne1 \end{cases}\)

  4. \(\int\frac{(Mx+N)dx}{(x^2+px+q)^n}=\int\frac{\frac M2(2x+p)+(N-\frac M2p)}{(x^2+px+q)^n}=\frac M2\int\frac{d(x^2+px+q)}{(x^2+px+q)^n}+(N-\frac M2p)\int\frac{dx}{(x^2+px+q)^n}\)

    • \(\int\frac{d(x^2+px+q)}{(x^2+px+q)^n}=\begin{cases}\ln(x^2+px+q)+C,n=1\\-\frac{1}{(k-1)(x^2+px+q)^{n-1}}+C,n\ne1\end{cases}\)

    • \(x^2+px+q=(x+\frac p2)^2+q-\frac{p^2}4=t^2+b^2\)，因此此类积分可转化为\(\int\frac{dt}{(t^2+b^2)^n}\)

      • \(J_n=\int\frac{dt}{(t^2+b^2)^n}=\begin{cases}\frac1b\arctan\frac tb+C,n=1\\\frac1{2(n-1)b^2}[\frac t{(t^2+b^2)^{n-1}}+(2n-3)J_{n-1}],n\ne1\end{cases}\)

某些被积表达式可以通过变元替换而有理化，例如：

（\(R(u,v)\)表示两个变量\(u,v\)的有理式）

  1. \(R(\sin x,\cos x)dx\)

    含三角函数的表达式可通过各种三角代换进行有理化，如万能代换。

  2. \(R(x,\sqrt{ax^2+bx+c})dx\)

    \(\sqrt{ax^2+bx+c}\)可化为\(\sqrt{u^2+-v^2}\)和\(\sqrt{v^2-u^2}\)，进而进行有理化。

  3. \(R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx\)

    若\(ad=bc\)，则\(a/c=b/d=k\)，\(R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx=R(x,\sqrt[n]k)dx\)

    若\(ad\ne bc\)，则可令\(t=\sqrt[n]\frac{ax+b}{cx+d}\)，则：

    \(x=\frac{dt^n-b}{a-ct^n}\)，\(dx=dt·\frac{n(ad-bc)t^{n-1}}{a-ct^n}\)

    原式=\(R(\frac{dt^n-b}{a-ct^n},t)dt·\frac{n(ad-bc)t^{n-1}}{a-ct^n}\)

  4. \(x^𝜆(a+bx^𝜇)^vdx\)

    设\(a,b\in\mathbb R\)，\(𝜆,𝜇,v∈Q\)，且\(a,b,𝜇,v\ne 0\)，为防止根号嵌套，可设\(x^𝜇=t\)，即\(x=\sqrt[\mu]t\)。

    此时\(dx=t^{\frac1𝜇-1}\frac{dt}𝜇\)。

    原式=\(\frac1𝜇t^{\frac{𝜆+1}{𝜇-1}}(a+bt)^vdt\)

    如果\(\frac{𝜆+1}𝜇\in\mathbb Z\)或\(\frac{𝜆+1}{𝜇+v}\in\mathbb Z\)，则可把该式转化为3中情形。

切比雪夫证明，除了上述情形以外，二项型微分式都不能被积分成有限形式。

3. 多元积分

多元积分

  1. 设\(I_k\)为\(k\)维闭区间\([\boldsymbol a,\boldsymbol b]\)，\(f:\mathbb R^k\rightarrow\mathbb R\)，
  2. 设\(f=f_k,f_{i-1}=\int_{a_i}^{b_i}f_i\mathrm dx_i,i=1,...,k\)，可得\(f_0=\int_{I_k}f(\boldsymbol x)\mathrm d\boldsymbol x\)。
  3. 该积分与自变量的积分次序无关，称为\(f\)在\(I_k\)的积分。
