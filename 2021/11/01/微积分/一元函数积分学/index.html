一元函数积分学

踏遍青山人未老，风景这边独好。

三 一元函数积分学

1. 原函数与不定积分

1.1 定义

  1. 原函数：

    设函数\(f\)在区间\(I\)上有定义，若函数\(F\)在区间\(I\)连续且可导，满足条件：\(\forall x\in I\)\(F’(x) = f(x)\)成立，

    称\(F\)是\(f\)的一个原函数，或者说\(F\)是微分形式\(f(x)dx\)的一个原函数。

    • 设函数\(f\)在区间\(I\)上有定义，若函数\(F\)是\(f\)的一个原函数，则\(\forall C\in R,F(x)+C\)也是\(f\)的原函数。

  2. 不定积分：

    设函数\(f\)在区间\(I\)上有定义，若函数\(F\)是\(f\)的一个原函数，则函数簇\(F(x)+C(C\in R)\)表示\(f\)的一切原函数，我们把这一函数簇称为函数\(f\)的不定积分，记为\(\int f(x)dx=F(x)+C\)。

    其中，\(f(x)\)称为被积函数，\(f(x)dx\)称为被积表达式，而\(\int\)是表示不定积分的符号。

    • 根据定义，有
      • \((\int f(x)dx)’=f(x)\)
      • \(d(\int f(x)dx)=f(x)dx\)
      • \(\int F'(x)dx=F(x)+C\)
      • \(\int dF(x)=F(x)+C\)
    • 不定积分的运算法则：
      • \(\int(f(x) + g(x))dx = \int f(x)dx +\int g(x)dx\)
      • \(\int\lambda f(x)dx = \lambda\int f(x)dx\)

1.2 存在定理

  1. 连续函数必有原函数。
  2. 含有第一类间断点和无穷间断点的函数在包含该间断点的区间内必无原函数。

1.3 基本不定积分表

  1. \(\int 0·dx = C\)

    \(\int1·dx = x + C\)

  2. \(\int x^\mu dx = \frac1{\mu + 1}·x^{𝜇 + 1} + C,𝜇\ne-1\)

    \(\int\frac1xdx=\ln|x|+C\)

    \(\int\frac1{x-a}dx=\ln|x-a|+C\)

  3. \(\int e^xdx=e^x+C\)

    \(\int a^xdx=\frac{a^x}{\ln a} + C\)

  4. \(\int\cos xdx = \sin x + C\)，\(\int \sin xdx = -\cos x + C\)

    \(\int \sec^2x·dx = \tan x + C\) ，\(\int\csc^2x·dx=-\cot x+C\)

    \(\int\sec\tan xdx=\sec x+C\)，\(\int\csc\cot xdx=-\csc x+C\)

  5. \(\int\tan xdx=-\ln|\cos x|+C\)，\(\int\cot xdx = \ln|\sin x| + C\)

    \(\int \sec xdx = \ln|\sec x + \tan x| + C\)，\(\int \csc xdx = \ln|\csc x - \cot x| + C\)

  6. \(\int\frac1{x^2+1}dx=\arctan x+C\)，\(\int\frac1{x^2+a^2}dx=\frac1a\arctan\frac xa + C\)

    \(\int\frac1{x^2-1}dx=\frac12\ln|\frac{x - 1}{x + 1}|+C\)，\(\int\frac1{x^2-a^2}dx=\frac1{2a}\ln|\frac{x - a}{x + a}| + C\)

  7. \(\int\frac1{\sqrt{1-x^2}}dx=\arcsin x + C\)，\(\int\frac1{\sqrt{a^2-x^2}}dx=\arcsin\frac xa + C\)

  8. \(\int\frac1{\sqrt{x^2+a^2}}dx=\ln (x+\sqrt{x^2+a^2})+C\)，\(\int\frac1{\sqrt{x^2-a^2}}dx=\ln|x+\sqrt{x^2-a^2}|+C\)

  9. \(\int \cosh xdx = \sinh x + C\)，\(\int \sinh xdx = \cosh x + C\)

1.4 积分法

换元积分法

  1. 一元函数微分表示的不变性：

    \(dG(u) = g(u)du\)，若把\(u\)换为可微函数\(u = u(v)\)，则有\(dG(u(v)) = g(u(v))du(v)\)。

    由此，从\(\int g(u)du = G(u) + C\)可以得到\(\int g(u(v))du(v) = G(u(v)) +C\)。

  2. 第一换元法（凑微分法）：\(\int f(x)dx=\int g(u(x))u'(x)dx=\int g(u(x))du(x)=G(u(x))+C\)

    • 用例：
      • \(\int g(ax+b)dx\)
      • \(\int g(x^k)x^{k-1}dx\)
      • \(\int g(\ln x)\frac1xdx\)
      • \(\int g(e^x)e^xdx\)
      • \(\int g(\cos x)\sin xdx,\int g(\sin x)\cos xdx\)
      • \(\int \tan xdx,\int \cot xdx\)

  3. 第二换元法：\(\int f(x)dx=\int f(u(v))du(v)=\int g(v)dv=G(v)+C=G(u^{-1}(x))+C\)

    • 用例：
      • \(\int\sqrt{x^2 + a^2}dx\)
      • \(\int\frac1{\sqrt{x^2\pm a^2}}dx\)
      • \(\int\frac1{\sqrt{a^2 - x^2}}dx\)
      • \(\int\frac1{x^2+a^2}dx\)

分部积分法

因为\(d(u(x)v(x)) = u(x)dv(x) + v(x)dv(x)\)，有\(\int u(x)dv(x)=u(x)v(x)-\int v(x)du(x)\)。

有理函数的积分

初等函数的导数一定是初等函数，但不少初等函数的不定积分并非初等函数，如：

\(\int e^{-x^2}dx\)，\(\int \sin x^2dx\)，\(\int \cos x^2dx\)，\(\int \frac{\sin x}xdx\)，\(\int\frac{\cos x}xdx\)，\(\int \frac x{\ln x}dx\)

一个不定积分不能用初等函数来表示，并非这个不定积分不存在。

事实上，任何连续函数\(f(x)\)都具有原函数。

有理函数：\(f(x) = \frac{P(x)}{Q(x)}=\frac{a_0+a_1x+...+a_nx^n}{b_0+b_1x+...+b_mx^m}\)：

此处\(P(x)\)和\(Q(x)\)都是实系数的多项式。若\(n<m\)，称\(f(x)\)为有理真分式，否则为有理假分式。

  • 利用多项式的除法，可以把有理假分式表示为多项式和有理真分式的和。

在实数范围内，一个多项式的不可约因式只可能是一次的或者二次的。设\(Q(x)\)的不可约因式分解如下：

\(Q(x)=(x-a_1)^{k_1}···(x-a_r)^{k_r}·(x^2+p_1x+q_1)^{k_1}...(x^2+p_sx+q_s)^{k_s}\)

其中\(a_i\)为实数，\(x^2+p_jx+q_j\)为不可约的实系数二次三项式，且其中每个因式互质，有\(p_i^2<4q_i\)。

有理真分式\(\frac{P(x)}{Q(x)}\)可以唯一地表示为以下形状的简单分式之和：

\(\int\frac{dx}{(x-a)^n}\)和\(\int\frac{(Mx+N)dx}{(x^2+px+q)^n}\)

该分解式称为真分式\(\frac{P(x)}{Q(x)}\)的简单分式分解或部分分式分解。

通过部分因式分解，求有理分式的不定积分的问题归结为计算以下两种类型积分：

  1. \(\int\frac{dx}{(x-a)^n}=\begin{cases}\ln|x-a|+C,n=1\\-\frac{dx}{(n-1)(x-a)^{n-1}}+C,n\ne1 \end{cases}\)

  2. \(\int\frac{(Mx+N)dx}{(x^2+px+q)^n}=\int\frac{\frac M2(2x+p)+(N-\frac M2p)}{(x^2+px+q)^n}=\frac M2\int\frac{d(x^2+px+q)}{(x^2+px+q)^n}+(N-\frac M2p)\int\frac{dx}{(x^2+px+q)^n}\)

    • \(\int\frac{d(x^2+px+q)}{(x^2+px+q)^n}=\begin{cases}\ln(x^2+px+q)+C,n=1\\-\frac{1}{(k-1)(x^2+px+q)^{n-1}}+C,n\ne1\end{cases}\)

    • \(x^2+px+q=(x+\frac p2)^2+q-\frac{p^2}4=t^2+b^2\)，因此此类积分可转化为\(\int\frac{dt}{(t^2+b^2)^n}\)

      • \(J_n=\int\frac{dt}{(t^2+b^2)^n}=\begin{cases}\frac1b\arctan\frac tb+C,n=1\\\frac1{2(n-1)b^2}[\frac t{(t^2+b^2)^{n-1}}+(2n-3)J_{n-1}],n\ne1\end{cases}\)
可有理化的被积表达式

某些被积表达式可以通过变元替换而有理化，例如：

（\(R(u,v)\)表示两个变量\(u,v\)的有理式）

  1. \(R(\sin x,\cos x)dx\)

    含三角函数的表达式可通过各种三角代换进行有理化，如万能代换。

  2. \(R(x,\sqrt{ax^2+bx+c})dx\)

    \(\sqrt{ax^2+bx+c}\)可化为\(\sqrt{u^2+-v^2}\)和\(\sqrt{v^2-u^2}\)，进而进行有理化。

  3. \(R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx\)

    若\(ad=bc\)，则\(a/c=b/d=k\)，\(R(x,\sqrt[n]{\frac{ax+b}{cx+d}})dx=R(x,\sqrt[n]k)dx\)

    若\(ad\ne bc\)，则可令\(t=\sqrt[n]\frac{ax+b}{cx+d}\)，则：

    \(x=\frac{dt^n-b}{a-ct^n}\)，\(dx=dt·\frac{n(ad-bc)t^{n-1}}{a-ct^n}\)

    原式=\(R(\frac{dt^n-b}{a-ct^n},t)dt·\frac{n(ad-bc)t^{n-1}}{a-ct^n}\)

  4. \(x^𝜆(a+bx^𝜇)^vdx\)

    设\(a,b\in\R\)，\(𝜆,𝜇,v∈Q\)，且\(a,b,𝜇,v\ne 0\)，为了防止根号嵌套，可设\(x^𝜇=t\)，即\(x=\sqrt[\mu]t\)。

    此时\(dx=t^{\frac1𝜇-1}\frac{dt}𝜇\)。

    原式=\(\frac1𝜇t^{\frac{𝜆+1}{𝜇-1}}(a+bt)^vdt\)

    如果\(\frac{𝜆+1}𝜇\in\Z\)或\(\frac{𝜆+1}{𝜇+v}\in\Z\)，则可把该式转化为3中情形。

    切比雪夫证明，除了上述情形以外，二项型微分式都不能被积分成有限形式。

2. 定积分

2.1 定义和性质

  1. 定积分概念的精确化是黎曼所作的贡献，因此我们也把这种积分称作黎曼积分。

  2. 分割：闭区间\([a,b]\)的一个分割，是指插入在\(a\)和\(b\)之间的有限个分点\(P:a=x_0<…<x_m=b\)。

    • 这些分点把\([a,b]\)分成\(m\)个闭子区间\([x_0,x_1]…[x_m-1,x_m]\)。

    • \(|P|=\max{(∆x_1,……,∆x_m)}\)称为分割\(P\)的模。

    • 在分割\(P\)的每一闭子区间上任意选取一点\(\xi_i∈[x_i-1,x_i]\)，\(i=1,2,…,m\)，这\(m\)个点称为相应于分割\(P\)的一组标识点，约定用单独一个字母\(\xi\)来表示它们。

    • 设函数\(f\)在闭区间\([a,b]\)上有定义，对\([a,b]\)的任何一个分割\(P\)和相应于这分割的任何一组标志点\(\xi\)，作和数\(\sigma(f,P,\xi)=\Sigma_1^mf(\xi_i)\Delta x_i\)。称这样的和数为函数\(f\)在闭区间\([a,b]\)上的积分和或黎曼和。

    • 如果闭区间\([a,b]\)的分割的序列\(\{P(n)\}\)满足\(\lim|P|=0\)，则称\(\{P(n)\}\)是一个无穷细分割序列。

  3. 黎曼积分：设函数\(f\)在闭区间\([a,b]\)上有定义，如果存在实数\(I\)，使得对于任意无穷细分割序列\(\{P(n)\}\)，不论相应于它的标志点组怎样选择，都有\(\lim\sigma(f,P,\xi)=I\)，那么我们就说函数\(f\)在\([a,b]\)上可积，并把\(I\)称为函数\(f\)在\([a,b]\)上的定积分。记为：\(\int_a^bf(x)dx=\lim_{|P|\rightarrow0}\sigma(f,P,\xi)=I\)。

    • 其中\(\int\)称为积分号，\(f(x)dx\)称为被积表达式，\(a\)和\(b\)称为积分下限和上限。

  4. 基本性质：

    • \(\int_a^bCdx=C(b-a)\)

    • 线性性质：

      \(\int_a^b(f(x)+g(x))dx=\int_a^bf(x)dx+\int_a^bg(x)dx\)

      \(\int_a^b𝜆f(x)dx=𝜆\int_a^bf(x)dx\)

    • 可加性：

      \(\int_a^cf(x)dx=\int_a^bf(x)dx+\int_b^bf(x)dx\)

    • 单调性：

      设\(a<b\)，函数\(f\)和\(g\)在\([a,b]\)上可积并且满足\(f(x)\le g(x)\)，则有\(\int_a^bf(x)dx\le\int_a^bg(x)dx\)

    • 积分中值定理：

      设\(a<b\)，函数\(f\)在\([a,b]\)可积(则\(f\)在\([a,b]\)上有界)，若\(∀x∈[a,b]\)有\(m\le f(x)\le M\)，则\(m(b-a)\le\int_a^bf(x)dx\le M(b-a)\)。

      特别的，如果\(f\)在\([a,b]\)连续，那么存在\(c∈[a,b]\)，使得\(\int_a^bf(x)dx=f(c)(b-a)\)。

2.2 存在定理

  1. 黎曼可积性：可以通过黎曼积分的定义计算积分值。也称为一元函数的常义可积性。

    • 常义，即区间有限、函数有界。区别于反常积分的区间无限、函数无界。

  2. 定积分存在定理：即一元函数的常义可积性。

    • 充分条件：

      • \(f(x)\)在\([a,b]\)上连续，则\(\int_a^b f(x)dx\)存在

      • \(f(x)\)在\([a,b]\)上单调，则\(\int_a^b f(x)dx\)存在

      • \(f(x)\)在\([a,b]\)上有界，且只有有限个间断点，则\(\int_a^b f(x)dx\)存在

    • 必要条件：有界。

2.3 变限积分

\(F(x)=\int_a^xf(t)dt\)为变上限的积分，同理也有变下限和上下限都变化的积分。

  • 性质：
    • \(f(x)\)在\(I\)上可积，则\(F(x)=\int_a^xf(t)dt\)在\(I\)上连续
    • \(f(x)\)在\(I\)上连续，则\(F(x)=\int_a^xf(t)dt\)在\(I\)上可导且\(F'(x)=f(x)\)
    • 若\(x=x_0\in I\)是\(f(x)\)唯一的跳跃间断点，则\(F(x)=\int_a^xf(t)dt\)在\(x_0\)处不可导
      • \(F'_-(x_0)=\lim_{x\rightarrow x_0^-}f(x)\)
      • \(F'_+(x_0)=\lim_{x\rightarrow x_0^+}f(x)\)
    • 若\(x=x_0\in I\)是\(f(x)\)唯一的可去间断点，则\(F(x)=\int_a^xf(t)dt\)在\(x_0\)处可导
      • \(F'(x_0)=\lim_{x\rightarrow x_0}f(x)\)
  • 求导公式：设\(F(x)=\int_{\varphi_1(x)}^{\varphi_2(x)}f(t)dt\)，则\(F'(x)=f(\varphi_2(x))\varphi_2'(x)-f(\varphi_1(x))\varphi_1'(x)\)。
    • 当\(f(t)\)中包含\(x\)，不能直接使用该求导公式。
  • 结论：
    • \(f(x)\)为可积的奇函数，则\(\int_a^xf(t)dt\)为偶函数。
    • \(f(x)\)为可积的偶函数，则\(\int_0^xf(t)dt\)为奇函数。

2.4 反常积分

  1. 无穷区间的反常积分：\(F(x)\)为\(f(x)\)在相应区间的一个原函数。
    • \(\int_a^{+\infin}f(x)dx=\lim_{x\rightarrow+\infin}F(x)-F(a)\)，极限存在即收敛，否则发散
    • \(\int_{-\infin}^bf(x)dx=F(b)-\lim_{x\rightarrow-\infin}F(x)\)，极限存在即收敛，否则发散
    • \(\int_{-\infin}^{+\infin}f(x)dx=\lim_{x\rightarrow+\infin}F(x)-\lim_{x\rightarrow-\infin}F(x)\)，两个极限都存在即收敛，否则发散
  2. \(f(x)\)的瑕点\(x_0\)：\(f(x)\)在\(x_0\)的邻域内。
  3. 无界函数的反常积分：\(F(x)\)为\(f(x)\)在相应区间的一个原函数。
    • \(x=a\)为瑕点，\(\int_a^bf(x)dx=F(b)-\lim_{x\rightarrow a^+}F(a)\)，极限存在即收敛，否则发散
    • \(x=b\)为瑕点，\(\int_a^bf(x)dx=\lim_{x\rightarrow b^-}F(b)-F(a)\)，极限存在即收敛，否则发散
    • \(x=c\in(a,b)\)为瑕点，\(\int_a^bf(x)dx=\int_a^cf(x)dx+\int_c^bf(x)dx\)，两积分都存在则收敛，否则发散
  4. 反常积分敛散性判别：比较法。
  5. 重要结论：
    • \(\int_0^1\frac1{x^p}\begin{cases}收敛，0\lt p\lt 1\\发散，p\ge1 \end{cases}\)
    • \(\int_1^{+\infin}\frac1{x^p}\begin{cases}收敛，p\gt 1\\发散，p\le1 \end{cases}\)

2.5 牛顿-莱布尼茨公式

设函数\(f\)在\([a,b]\)连续，如果存在函数\(F\)，它在\([a,b]\)连续且可导，并且满足\(F’(x)=f(x)\)，那么函数\(f\)在\([a,b]\)可积，且\(\int_a^bf(x)dx=F(b)-F(a)\)。

  • 基于牛顿-莱布尼茨公式，求定积分可转化为求不定积分。

连续可微：

  • 如果函数\(f\)在\((a,b)\)上每一点可导，并且导函数在\((a,b)\)连续，那么我们说函数\(f\)在\((a,b)\)连续可微。

  • 如果函数\(f\)在\([a,b]\)上每一点可导(在\(a\)右可导，在\(b\)左可导)，并且导函数在\([a,b]\)连续，那么我们说函数\(f\)在\([a,b]\)连续可微。

换元法

设函数\(𝜑∈C^1[𝛼,𝛽]\)，\(𝜑(𝛼)=a\)，\(𝜑(𝛽)=b\)，\(𝜑((𝛼,𝛽))⊂(a,b)\)，如果\(f\)在\([a,b]\)连续，那么\(\int_a^bf(x)dx=\int_\alpha^\beta f(𝜑(t))𝜑’(t)dt\)

分部积分

设函数\(u,v\in C^1[a,b]\)，则\(\int_a^bu(x)v’(x)dx=u(x)v(x)|_a^b-\int_a^bv(x)u’(x)dx\)

常用结论

  1. \(f(x)\)为连续偶函数，则\(\int_{-a}^af(x)dx=2\int_0^af(x)dx\)。

  2. \(f(x)\)为连续奇函数，则\(\int_{-a}^af(x)dx=0\)。

  3. \(f(x)\)是周期为\(T\)的连续函数，则\(\forall a,\int_a^{a+T}f(x)dx=\int_0^Tf(x)dx\)。

  4. \(f(x)\)为连续函数，则\(\int_a^bf(x)dx=\int_a^bf(a+b-x)dx\)

  5. 华莱士公式：

    \(\int_0^{\frac\pi2}\sin^nxdx=\int_0^{\frac\pi2}\cos^nxdx=\begin{cases}\frac{(n-1)!!}{n!!},n为大于1的奇数\\\frac{(n-1)!!}{n!!}\frac\pi2,n为正偶数 \end{cases}\)

    • 派生：
      • \(\int_0^{\pi}\sin^nxdx=2\int_0^{\frac\pi2}\sin^nxdx\)
      • \(\int_0^{\pi}\cos^nxdx=\begin{cases}0,n为正奇数\\ 2\int_0^{\frac\pi2}\sin^nxdx,n为正偶数\end{cases}\)
      • \(\int_0^{2\pi}\cos^nxdx=\int_0^{2\pi}\sin^nxdx=\begin{cases}0,n为正奇数\\ 4\int_0^{\frac\pi2}\sin^nxdx,n为正偶数\end{cases}\)

3. 应用

几何计算

  1. 平面图形面积：

    • 直角坐标系：\(y=f(x),y=g(x)\)与\(x=a,x=b\)围成的图形面积：\(\int_a^b|f(x)-g(x)|dx\)
    • 极坐标系：\(r=\rho_1(\theta),r=\rho_2(\theta)\)与\(\theta=\alpha,\theta=\beta\)围成的图形面积：\(\frac12\int_\alpha^\beta|\rho_1^2(\theta)-\rho_2^2(\theta) |d\theta\)
    • 参数方程：转换为直角坐标系并换元

  2. 旋转体体积：

    • \(y=f(x),x=a,x=b\)及\(x\)轴围成的图形绕\(x\)轴旋转一周所得旋转体体积：\(\int_a^b\pi f(x)^2dx\)

    • \(y=f(x),x=a,x=b\)及\(x\)轴围成的图形绕\(y\)轴旋转一周所得旋转体体积：\(2\pi\int_a^bx|f(x)|dx\)

    • 平面曲线\(y=f(x)\)绕定直线\(Ax+By+C=0\)旋转一周所得旋转体体积：

      \(\frac\pi{(A^+B^2)^{\frac32}}\int_a^b[Ax+Bf(x)+C]^2|Af'(x)-B|dx\)

  3. 函数平均值：\(y=f(x)\)在\([a,b]\)上的平均值为\(\frac1{b-a}\int_a^bf(x)dx\)

  4. 曲边梯形的质心：\(y=f(x),x=a,x=b\)所围曲边梯形质心坐标：

    \(x=\frac{\int_a^bxf(x)dx}{\int_a^bf(x)dx}\)，\(y=\frac{\frac12\int_a^bf(x)^2dx}{\int_a^bf(x)dx}\)

  5. 平面曲线的弧长

    • \(y=f(x),x\in[a,b]\)：\(\int_a^b\sqrt{1+f'(x)^2}dx\)
    • \(x=x(t),y=y(t),t\in[\alpha,\beta]\)：\(\int_\alpha^\beta\sqrt{x'(t)^2+y'(t)^2}dt\)
    • \(r=\rho(\theta),\theta\in[\alpha,\beta]\)：\(\int_\alpha^\beta\sqrt{[\rho(\theta)^2+\rho'(\theta)^2]}dx\)

  6. 旋转曲面的面积：

    • \(y=f(x),x\in[a,b]\)绕\(x\)轴旋转：\(2\pi\int_a^b|f(x)|\sqrt{1+f'(x)^2}dx\)
    • \(x=x(t),y=y(t),t\in[\alpha,\beta]\)绕\(x\)轴旋转：\(2\pi\int_\alpha^\beta|y(t)|\sqrt{x'(t)^2+y'(t)^2}dt\)
    • \(r=\rho(\theta),\theta\in[\alpha,\beta]\)：\(2\pi\int_\alpha^\beta|\rho(\theta)\sin\theta|\sqrt{[\rho(\theta)^2+\rho'(\theta)^2]}d\theta\)

物理计算

  1. 变力按直线做功：\(W=\int_a^bF(x)dx\)
  2. 抽水做功：水面距离容器口高度\(x\)，对应水平截面面积\(A(x)\)，\(W=\rho g\int_a^bxA(x)dx\)
  3. 静水压力：平板两曲边距水距离为\(f(x),g(x)\)，\(P=\rho g\int_a^bx(f(x)-g(x))dx\)
